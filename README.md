# Assignment_1_AdvancedOperatingSystems
The repository for first assignment of AOS
# feature useage
## Task 1
This is a resource management system, it displays CPU and Memory useage, aswell as having the options to terminate processes and displaying top 10 CPU hungry resources
The program is simple just select one of the 5 options, 
Terminate a process will require a PID of the process you want to terminate, However if the Process has a Priority (or Nice) value below 0 it will not allow you to terminate

## Task 2
This is a Job Scheduler using Python, this will allow you to create "Fake" processes with a User ID, Process name, Time required and Priority, the program will prmpt you when you use this.
The Process Jobs option will clear out all of the jobs using a Priority first methodology, meanign highest priority Jobs will be done first

## Task 3
This task is an Assignment submission tool, it will create a submitted folder where all submitted files will be located once submitted.
Restrictions for submittable files is 50MB max size and only .pdf and .docx files are accepted.
This Program will also check i that files already been submitted
Both Bash and Python examples are avalible
